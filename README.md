## Pages and Functionality

### Contacts

This page allows you to manage your contacts. You can:
- Add a new contact by filling out the form, and the contact will be added to the list.
- Edit or delete an existing contact.
- Click on a contact's name to view the contact details page.

### Charts and Maps

This page displays COVID-19 data for different countries. It includes:
- A line graph showing the fluctuation of cases over time.
- A map with markers indicating the country name, total number of active cases, recovered cases, and deaths, displayed as a popup.
